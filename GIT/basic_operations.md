Git Basic Commands
==================

Adding, committing
------------------
```
git add -A
git commit -m '___'
```

Initializing a repo
-------------------
```
git init
git add .
git commit -m "abc"
git remote add origin <remote_repository_url>
git push -u origin master
```

Deleting branches
-----------------
$ git push -d <remote_name> <branch_name>
$ git branch -d <branch_name>
